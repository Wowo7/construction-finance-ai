# pctd - 049-nextjs-supabase-ai-chat

## Summary
Construction Finance AI Chat - an AI-powered assistant that lets construction PMs query project budgets in plain English. Built with Next.js 15, Supabase, and OpenRouter (Claude Sonnet 4). Features 5 AI tools that call Supabase RPC functions to retrieve budget data across 3 demo projects with 44 packages and 10 trades.
